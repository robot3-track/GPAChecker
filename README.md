# GPA/Grade Checker
This project is a free open-source GPA/Grade checker for students to use. Students can input weight of subjects and potential grades to test possible outcomes.

# Features
The project includes graphs that depict waht is needed to reach the student's goals, as well as predictions based off potential inputs.

# API Keys Used
Gemini API Key - For copy/pasting your class syllabus to create automatic grade weighting format.

# Try it now!
Try the live deployment at: https://gpacheck.vercel.app

# How to setup youself
First, fork this repository and make sure to give credit to Robot3-track. Then, connect to vercel for Next.js webapp deployment. Get your own Gemini/AI API Key and set it as an enviromental variable in the enviornmental variables tab within Vercel Deployments.
